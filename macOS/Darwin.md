# Darwin macOS

### Package management - Homebrew
`/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"`

`brew config # to verify`

What is installed? `grep 'Installed' /private/var/log/install.log`
